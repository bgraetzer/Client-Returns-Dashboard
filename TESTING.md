// README addition: How to run tests

## Running Tests

1. Ensure you have Node.js installed (https://nodejs.org/).
2. Install Mocha globally if not already installed:
   npm install -g mocha
3. In this folder, run:
   mocha *.test.js

This will run all test files and report results in the terminal.

You can add more tests by creating new `.test.js` files.
